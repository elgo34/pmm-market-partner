# PMM Market Partner — Guide d'utilisation

## Ce que fait cet agent

Expert marche factuel qui collecte, structure et confronte les donnees du marche photo US. Il opere sur 4 niveaux : opportunites, positionnement, messaging, validation GTM.

## Ce qu'il ne fait pas

Il ne prend pas de decisions, ne recommande pas, n'interprete pas. Tout est source, date, factuel.

---

## Commandes

### Par niveau d'analyse

| Commande | Ce qu'elle fait |
|---|---|
| `/opportunities` | Analyse marche : sizing, tendances, segmentation, mouvements |
| `/positioning` | Mapping value props, espaces disponibles, confrontation |
| `/messaging` | Collecte messaging, patterns, audit maquettes |
| `/validate-gtm` | Confrontation du plan GTM contre les donnees et le contexte |

### Transversales

| Commande | Ce qu'elle fait |
|---|---|
| `/competitor [nom]` | Deep dive sur un concurrent (toutes les sources, iteratif) |
| `/benchmark [feature]` | Benchmark comparatif sur une feature (tous les concurrents) |
| `/audit` | Audit UX/wording d'une maquette vs positionnement et concurrents |
| `/help-pmm` | Ce guide |

---

## Comment l'utiliser

- **Entree libre** — entre a n'importe quel niveau. L'agent challengera si les niveaux amont ne sont pas couverts.
- **3 postures** — selon ta demande :
  - "Fais-moi une etude sur X" → faits bruts
  - "Comment se positionne-t-on vs les autres ?" → confrontation
  - "Que ne font pas les concurrents ?" → detection de patterns
- **Sois precis** — plus ton input est clair, plus l'analyse est ciblee. L'agent poussera pour clarifier si c'est flou.
- **Output Notion** — l'agent te demandera ou stocker les resultats dans Notion.
