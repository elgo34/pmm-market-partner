## Concurrents directs

Apps et services d'impression photo :

- **Shutterfly** — impression photo, livres, cartes, cadeaux. Web + app native.
- **Snapfish** — impression photo, livres, cadeaux. Web + app native.
- **FreePrints** — impression photo gratuite (modele freemium). App native principalement.
- **Chatbooks** — livres photo automatises. App native principalement.
- **Mixbook** — livres photo, cartes. Web principalement.
- **Artifact Uprising** — impression photo premium. Web + app native.
- **Nations Photo Lab** — impression photo pro/consumer. Web principalement.
- **Mpix** — impression photo qualite pro. Web principalement.

## Concurrents adjacents

Acteurs qui touchent a la photo sans etre concurrents frontaux :

- **Google Photos** — stockage, organisation, edition, impression (integration). Web + app native.
- **Apple Photos** — stockage, organisation, edition, impression (integration). App native + Mac.
- **Amazon Photos** — stockage, impression. Web + app native.
- **Canva** — creation graphique, de plus en plus photo. Web + app native.
- **Adobe Lightroom / Photoshop Express** — edition photo. Web + app native.
- **VSCO** — edition photo, communaute. App native.
